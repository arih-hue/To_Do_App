# Flutter To-Do App

A beautifully simple, beginner-friendly To-Do application built with Flutter. This app helps users manage their daily tasks with a clean UI, swipe-to-delete functionality, and persistent local storage so you never lose your data.

## 🚀 Features

* **Add Tasks:** Easily create new tasks using a custom dialog box.
* **Track Progress:** Check and uncheck tasks as you complete them.
* **Swipe to Delete:** Slide any task to the left to quickly delete it.
* **Local Storage:** Tasks are saved locally on your device using Hive, so your list is right there when you close and reopen the app.

## 🛠️ Tech Stack & Packages

* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** Dart
* **Local Database:** `hive_flutter` (Fast, lightweight, NoSQL database)
* **UI Components:** `flutter_slidable` (For the swipe-to-delete action)

## 🎨 UI Highlights
* Custom gradient AppBar.
* Floating Action Button for quick task entry.
* Clean, minimalist color palette using cool blues and dark greys.
